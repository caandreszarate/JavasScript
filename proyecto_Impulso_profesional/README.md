# Impulso Profesional - Plataforma de Empleabilidad

## Descripción
Plataforma web moderna para conectar jóvenes profesionales con empresas en Guinea Ecuatorial, inspirada en [Impulso Profesional](https://impulsoprofesionalge.org/).

## Características
- 🎯 Dashboard personalizado para usuarios registrados
- 📱 Diseño responsive para móviles y tabletas
- 🏢 Portal para empresas y organizaciones
- 👥 Sistema de perfiles profesionales
- 📊 Panel de administración
- 🔐 Sistema de autenticación seguro
- 📝 Formularios de postulación inteligentes

## Tecnologías
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: PHP 8.0+
- **Base de Datos**: MySQL (Hostinger)
- **UI/UX**: Diseño moderno con animaciones suaves
- **Responsive**: Mobile-first design

## Instalación
1. Clonar el repositorio
2. Configurar la base de datos MySQL
3. Configurar las credenciales en `backend/config/database.php`
4. Subir archivos al servidor Hostinger
5. Configurar el dominio
